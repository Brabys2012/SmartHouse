SmartHouse
==========
Проект, реализующий технологию "Умный дом" (управление жизнедеятельностью дома).

Описание каталога:
- Hardware - содержит проектную документацию аппаратной части проекта.
- Software - содержит программную документацию и ее реализацию.
- Software\Smart House - исходные коды программного комплекса.
- Software\Smart House\ControlsNOBD - исходные коды виртуального контроллера.
- Software\Smart House\Terminal - исходные коды программы терминала для работы по последовательному порту.
